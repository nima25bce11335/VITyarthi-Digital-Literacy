# VITyarthi-Digital-Literacy
Digital literacy is the ability to use digital devices, the internet, and online tools effectively and safely. It includes skills like finding and evaluating information, communicating online, and protecting privacy. It helps people learn, work, and participate responsibly in today’s digital world.
